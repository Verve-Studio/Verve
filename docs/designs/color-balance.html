Design decisions
Gradient slider tracks: Slider tracks use a full cyan→red / magenta→green / yellow→blue gradient in place of the standard accent-fill pattern. The full gradient shows at all times; the white dot thumb conveys the current position. The white tick at 50% (zero) is rendered with slightly higher opacity on the gradient to remain visible across all hues.
Reset scope: The Reset button resets only the currently active tab's three sliders to 0. Preserve Luminosity is not affected by Reset (it is a global setting, not per-tab).
Tab active underline: Active tab is indicated by a 2px bottom border in accent blue plus a slightly lighter background blending into the panel body. Inactive tabs share the darker #252525 background. Vertical 1px dividers between tabs use margin-bottom: -1px to merge the active tab into the body without a gap.
Label widths: Panel is 252px (vs 236px for B/C) to accommodate the dual left+right label design. Left labels are right-aligned (flush against slider), right labels are left-aligned. "Magenta" (longest label) gets 46px; right labels (Red/Green/Blue) get 30px.
Preserve Luminosity placement: Checkbox is outside all three tab panels so it persists visually when switching tabs. Separated from slider rows by a 1px border-top. Default: checked (per spec).
Canvas preview: CSS hue-rotate approximation driven by the active Midtones CR slider value. Positive CR (toward Red) rotates hue toward warm; negative (toward Cyan) adds cool cast. This is a visual prototype approximation only — the real algorithm uses tonal-range weighting.
