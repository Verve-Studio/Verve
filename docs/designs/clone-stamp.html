Cursor behaviour Native cursor is hidden (cursor:none). The circle div is driven imperatively via ref on every onHover call — no React state updates. White border + dark box-shadow ensures visibility on all backgrounds. The center crosshair helps precise source alignment. The source marker uses the same composite rendering approach.
1
Options bar — left side Size, Hardness, Opacity follow the same SliderInput + number field pattern as Brush and Eraser. Hardness 0% = fully feathered, 100% = hard edge.
2
Aligned / Sample All Layers checkboxes Inline checkboxes using the optCheckLabel pattern from Eraser. "Aligned" defaults on; "Sample All Layers" defaults off per spec.
3
Source status pill (options bar — right side) Small pill at the trailing end of the options bar. Grey/empty when no source; accent-blue with dot when source is set. Updates dynamically.
4
Pre-source state (canvas) Cursor is a plain crosshair (no circle). A small tooltip reads "Alt + click to set source point". Attempting to paint does nothing.
5
Source marker Rendered directly on canvas (not a DOM overlay). White crosshair arms + small circle, with dark box-shadow outline. Tracks brush offset in Aligned mode; stays fixed in Non-Aligned mode.
6
Offset dashed line (Aligned mode) A semi-transparent dashed line connecting the source marker to the brush cursor, visible while source is set. Helps the user understand the clone offset at a glance. Hidden in Non-Aligned mode.
7
Layer panel — "source" badge The layer locked as clone source at Alt-click time receives a small accent-blue badge labelled "source". Cleared when the source layer is deleted or a new Alt-click resets it.
8
Source-deleted notification A subtle danger-tinted toast bar at the bottom of the workspace (not a modal). Auto-dismisses after ~4 s. Mirrors the notification style used for other transient errors.
