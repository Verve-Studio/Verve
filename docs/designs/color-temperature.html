Design Decisions & Flags
Slider Tracks

Temperature: full-range gradient from cool blue (#5599ff) left to warm orange (#ff9933) right. The gradient replaces the standard accent-fill pattern because the color itself communicates the effect direction — no fill indicator is needed. design decision

Tint: green (#44bb44) → magenta (#dd44dd). Same rationale.

The zero-tick line is rendered in white at 25% opacity so it remains subtly visible against both coloured tracks.

Label Width

Label column widened to 72px (vs. 62px in Brightness/Contrast) to fit "Temperature" without truncation. Tint is shorter so the extra width leaves comfortable whitespace. design decision

Separator between the two rows is the standard 1px #191919 divider — same as Brightness/Contrast.

Canvas Preview

Preview image is a warm/cool landscape gradient (sky → earth tones) to give visible response to both Temperature and Tint changes. Temperature shift is approximated via CSS sepia() + hue-rotate(); Tint via hue-rotate() + saturate(). prototype approximation

In the real implementation the WASM layer applies per-pixel channel arithmetic.
