7 — Keyboard Shortcuts
W Activate tool / cycle with Magic Wand
Enter Commit point-prompt selection
Esc Cancel / clear point prompts
Backspace Remove last point prompt + re-run inference
Alt Add negative prompt point while in Point mode
Shift Add mode (hold to temporary switch selection mode)
Alt+Shift Intersect mode (hold)
Ctrl+Alt Subtract mode (hold)
