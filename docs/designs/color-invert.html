Design Decisions & Flags
Minimal Panel

Per spec, the panel contains no sliders, checkboxes, or editable controls. The body is a single descriptive sentence that confirms what the adjustment does and reminds the user that the eye toggle is the control. spec driven

Panel width reduced to 220px from the standard 236px to avoid oversized empty space. design decision

No Reset Button

The footer has no Reset button — there are no parameters to reset. Spec explicitly states no configurable parameters exist. The footer retains only the parent-layer indicator for context. spec driven

This differentiates it visually from parameter-bearing adjustment panels, reinforcing the "parameter-free" nature of this adjustment.

Eye Toggle as Primary Control

The prototype's Layer Panel shows a clickable eye button on the adjustment row. Clicking it toggles CSS filter: invert(1) on the preview canvas, demonstrating the real-time enable/disable UX described in the spec. key interaction

The panel description text explicitly mentions the eye toggle so first-time users discover it immediately.
