Design Decisions & Flags
Range Selector: Dropdown vs. Tab Strip

The spec says "dropdown or horizontal tab strip." A dropdown was chosen because 9 tab labels would overflow the 248px panel width even at 10px. The dropdown pattern also matches Photoshop Selective Color exactly. design decision

A colour swatch dot is placed beside the dropdown to give instant visual context for which range is active without reading the label. addition beyond spec

CMYK Slider Tracks

Each slider uses a characteristic colour gradient: grey→cyan, grey→magenta, grey→yellow, dark-grey→black. The grey left anchor represents zero channel content; the coloured right anchor represents maximum addition. design decision

This gives a visual mnemonic for what each slider does, consistent with the Temperature/Tint approach in the sibling panel.

Reset Button Scope

The spec says "Reset button (resets current range only)." The button label is always "Reset [RangeName]" (e.g. "Reset Reds") so the user knows the scope before clicking. design decision

Panel height is deliberately taller than the two-slider panels to accommodate the 4-slider layout plus method toggle without crowding. Panel width increased to 248px to give "Magenta" label enough room.
