Design notes
Panel structure
Three logical groups separated by row-sep dividers:
Group 1: Color + Opacity — appearance of the stroke.
Group 2: Thickness + Position + Softness — geometry of the stroke.
Group 3 (footer): Layer indicator + Reset button.

No Blend Mode dropdown. No Knockout checkbox. The Outline effect always composites the stroke using Normal blend mode behind the source layer.
Differences from Drop Shadow / Glow
No X/Y Offset — the outline is positional relative to the source silhouette only.
No Blend Mode — always Normal, always behind source pixels.
No Knockout — the stroke is inherently composited under the source, so knockout is implicit.
Position dropdown (Outside / Inside / Center) replaces the Spread slider from Drop Shadow. Softness range is 0–50 px (vs. 0–100 px in Glow/Drop Shadow).
Thickness input constraints
Range: 1–100 px. Integer-only. Non-integer input is rounded to nearest integer on blur (e.g., 3.7 → 4). Values below 1 clamp to 1; values above 100 clamp to 100. The slider uses min="1" so the handle never reaches the far-left end. Slider fill pct = (value − 1) / 99.
Reset button
Restores all five controls to their defaults:
Color = #FF0000FF, Opacity = 100%, Thickness = 3, Position = Outside, Softness = 0.

Placed in the footer row alongside the parent layer indicator. Same reset-btn style as Drop Shadow and Glow — transparent background, border, muted text color, hover state.
