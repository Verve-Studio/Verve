Hue group order reference

The canonical left-to-right sort order for the swatch grid. Each column is one hue family; neutral grays sit at the trailing end. Within each family: dark (top) → light (bottom).

Hue groups (left → right): Reds · Oranges · Yellows · Yellow-greens · Greens · Cyan-greens · Cyans · Blues · Royal blues · Blue-violets · Magentas/Purples · Red-magentas — then Neutrals (black → white). No dividers or labels appear in the actual panel; grouping is communicated through proximity alone.
