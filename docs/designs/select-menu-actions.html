Item reference
Item Shortcut Effect
All Ctrl+A / Cmd+A Sets pixel selection to full canvas rectangle (0,0)→(W−1,H−1). Shows marching ants at canvas border.
Deselect Ctrl+D / Cmd+D Clears pixel selection mask to null. Marching ants disappear. Full layer is editable.
— separator —
All Layers Alt+Ctrl+A / Alt+Cmd+A Sets selectedLayerIds to all layer ids (incl. group children). All rows highlighted in Layers panel.
Deselect Layers — Sets selectedLayerIds to []. No layer rows highlighted. Active layer indicator unchanged.
— separator —
Invert Selection Ctrl+Shift+I Existing behavior. Inverts the pixel selection mask. Unaffected by this feature.
