Amber warning for any conversion that could clamp HDR data (rgba32f → rgba8 or rgba32f → indexed8).

RGB/8 → Indexed/8 (adj. suspended)
Convert Color Mode
RGB/8
→
Indexed/8

Convert document to Indexed/8? This will convert all raster layer data. This operation can be undone.

All adjustment layers will be suspended in Indexed/8 mode.

Any conversion targeting Indexed/8 always shows the adjustment-suspension warning. Palette-quantization loss is implicit in the description text.

RGB/32 Float → Indexed/8 (both warnings)
Convert Color Mode
RGB/32 Float
→
Indexed/8

Convert document to Indexed/8? This will convert all raster layer data. This operation can be undone.

Out-of-range (HDR) pixel values will be clamped to the 0–255 range.
All adjustment layers will be suspended in Indexed/8 mode.

Worst-case: both warnings stack. HDR clamp fires because source is rgba32f; adjustment suspension fires because target is Indexed/8. Declaration order is fixed: HDR clamp first, adjustment suspension second.

Surface 4 — Disabled Tool Treatment (Indexed/8 mode)
Toolbar — RGB/8 (all tools enabled)

In RGB/8 and RGB/32F modes: all tools fully enabled. Active tool has a surface-selected background. Standard icon color.

Toolbar — Indexed/8 (5 tools disabled)

In Indexed/8: Brush, Gradient, Clone Stamp, Dodge, and Burn are disabled at opacity 0.32. cursor: not-allowed

On hover over a disabled tool, a tooltip appears showing the tool name and the reason: "Requires RGB/8 or RGB/32F mode."
