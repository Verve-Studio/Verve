Color Dithering Setup

Color Dithering maps composited pixels to your document palette using classic dithering algorithms — Bayer ordered patterns or Floyd-Steinberg error diffusion. For the best retro look, set up your palette first and optionally add a Reduce Colors layer below.

1
Configure the document palette Optional

The dithering algorithm quantizes to colors in the Swatches panel. If your palette is empty or not yet configured, set it up first for best results.

2
Add a Reduce Colors layer below Optional

A Reduce Colors adjustment in Map to Palette mode constrains source colors before dithering, producing cleaner retro palette fidelity.
