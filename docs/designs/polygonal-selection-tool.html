Keyboard Shortcuts
L Activate Polygonal Lasso tool
Esc Cancel in-progress polygon (discard all vertices)
⌫ Remove last placed vertex (Backspace / Delete)
dbl-click Close polygon (connects last vertex to first)
Shift Add to selection (held on first click)
Alt Subtract from selection (held on first click)
Shift + Alt Intersect with selection (held on first click)
Design Notes
1
Origin anchor dot A permanent filled circle is drawn at the first vertex for the entire polygon session. In normal drawing state it is a plain white dot. In near-close state (~12 px snap radius) it grows, gains an accent-blue ring and glow to unambiguously signal the polygon will close on next click.
2
Rubber-band line A dashed white line (dark backing stroke for visibility on light content) trails from the last placed vertex to the live cursor position on every pointer-move. Disappears immediately on close or cancel. Never composited into the pixel canvas.
3
Mode buttons Four compact icon buttons (New / Add / Subtract / Intersect) at the far left of the options bar after the tool badge. The active button is tinted accent-blue. Modifier keys at draw-time always override the bar setting for that polygon session; the bar reflects the persistent default.
4
Marching ants After commit, a CSS-animated dashed polygon outline (white stroke over dark backing, 8/4 dash pattern, ~0.6 s cycle) appears around the committed region. The selection interior receives a faint accent-blue tint (7% opacity) to aid visibility of the selected area.
5
Backspace / undo vertex Each Backspace removes the most recently placed vertex; the rubber-band retracts to the previous point. A second Backspace on a one-vertex polygon cancels entirely — same effect as Escape. Keys only respond while a polygon is in progress.
6
Double-click to close The double-click position is not added as a vertex. The final segment is drawn from the last placed vertex directly to the first anchor point, regardless of where the double-click landed. This prevents accidental degenerate slivers.
